# 康托展开

<hr />

在 C++ 算法竞赛（如 ACM/ICPC、Codeforces、LeetCode 等）中，**康托展开** 是一个非常重要的数学工具。

简单来说，**康托展开是一个全排列到一个自然数的双射映射**。它可以让你把一个 1 到 n 的排列，唯一地转换成一个整数（通常代表该排列在字典序中的排名），并且也能逆推回来。

## 1.核心概念

康托展开的作用是：**给定一个排列，求它在全排列中的字典序排名（从 0 开始）**。

例如，对于 ``1,2,3`` 的全排列：

- 排列 ``1 2 3`` 的排名是 **0**
- 排列 ``1 3 2`` 的排名是 **1**
- 排列 ``2 1 3`` 的排名是 **2**
- …
- 排列 ``3 2 1`` 的排名是 **5**
