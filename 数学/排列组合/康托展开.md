# 康托展开

<hr />

在 C++ 算法竞赛（如 ACM/ICPC、Codeforces、LeetCode 等）中，**康托展开** 是一个非常重要的数学工具。

简单来说，**康托展开是一个全排列到一个自然数的双射映射**。它可以让你把一个 1 到 n 的排列，唯一地转换成一个整数（通常代表该排列在字典序中的排名），并且也能逆推回来。

## 1.核心概念

康托展开的作用是：**给定一个排列，求它在全排列中的字典序排名（从 0 开始）**。

例如，对于 ``1,2,3`` 的全排列：

- 排列 ``1 2 3`` 的排名是 **0**
- 排列 ``1 3 2`` 的排名是 **1**
- 排列 ``2 1 3`` 的排名是 **2**
- …
- 排列 ``3 2 1`` 的排名是 **5**

<hr />

## 2.数学原理

假设我们有一个排列 $A=\{a_{1},a_{2},...,a_{n}\}$ ，我们要计算它的康托展开值 $X$（即排名）。

公式如下：

$X=a_{1}\times (n-1)!+a_{2}\times (n-2)!+...+a_{n-1}\times 1!+a_{n}\times 0!$

这里的系数 $a_{i}$ （通常记为$c_{i}$或temp）**不是排列中的元素本身**，而是指：

**在当前位置$a_{i}$的右边，有多少个比$a_{i}$小的数（或者是还未使用的数中比$a_{i}$小的个数）**。

<hr />

## 3.手动计算示例

以排列 ``3 2 1`` 为例（n=3）：

1.**第一位是 3**：

- 在它右边的数是 ``{2, 1}``。
- 比 3 小的数有 2 个（1 和 2）。
- 系数 $c_{1}$=2
- 贡献值：2×(3−1)!=2×2!=4。

2.**第二位是 2**：

- 在它右边的数是 {1}。
- 比 2 小的数有 1 个（1）。
- 系数 $c_{2}$=1。
- 贡献值：1×(3−2)!=1×1!=1。

3.**第三位是 1**：

- 它是最后一位。
- 比 1 小的数有 0 个。
- 系数 $c_{3}$=0。
- 贡献值：0×0!=0。

**最终结果**：4+1+0=5。

这表示 ``3 2 1`` 是第 5 个排列（从 0 开始数）

## 4.C++ 代码实现（朴素版）

最直观的方法是用循环模拟，每次统计右边比当前数小的个数。时间复杂度为 $O(n^{2})$。适用于 n 较小的情况（如 $n≤10000$）。
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

// 预处理阶乘，注意阶乘增长非常快，n=20 左右就会爆 long long
long long factorial[25]; 

void init_factorial(int n) {
    factorial[0] = 1;
    for (int i = 1; i <= n; i++) {
        factorial[i] = factorial[i-1] * i;
    }
}

// 康托展开
// s: 存储排列的数组（假设内容为 1~n 的整数）
// n: 数组长度
long long cantor(vector<int> s, int n) {
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        // 统计 s[i] 右边有多少个比它小的数
        int cnt = 0;
        for (int j = i + 1; j < n; j++) {
            if (s[j] < s[i]) cnt++;
        }
        ans += cnt * factorial[n - 1 - i];
    }
    return ans;
}

int main() {
    init_factorial(10);
    vector<int> p = {3, 2, 1};
    cout << "Rank: " << cantor(p, 3) << endl; // 输出 5
    return 0;
}
```
