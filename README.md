## 使用方式

- 建议用Vistual Studio打开本仓库，然后在左侧的侧边栏中选择 ``Algorithm`` 文件夹，即可查看相关内容。
- 不要直接在VS Code里面运行代码，配置环境比较麻烦。
- 可以直接将模板粘贴到对应的题目中，然后进行编译和运行。
