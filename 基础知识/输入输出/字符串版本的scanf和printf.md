## 字符串版本的 scanf 和 printf

在 C++中，`sscanf` 和 `sprintf` 是 C 语言标准库（`<cstdio>` 或 `<stdio.h>`）中的两个极其强大的函数。

你可以把它们理解为 “**字符串版本的 `scanf` 和 `printf`**”。

- **普通 `scanf`/`printf`**：数据流向是 **键盘 <-> 内存变量**。
- **`sscanf`/`sprintf`**：数据流向是 **字符串 <-> 内存变量**。

**一、 `sprintf`：把数据“写”进字符串（格式化输出）**<br />
**作用**：将各种类型的数据（整数、浮点数等）按照指定的格式，拼接/转换成一个字符串。<br />

**函数签名：**

```cpp
int sprintf(char *str, const char *format, ...);
```

- `str`：目标字符串数组（用来装结果的容器）。
- `format`：格式控制符（和 `printf` 用法一模一样，如 `%d`, `%f`, `%s`）。
- `...`：要填入的变量。
- 返回值：成功写入的字符个数（不包括结尾的 `\0`）。

**1. 基础用法：数字转字符串**
这是 `sprintf` 最常见的用途，可以替代 C++ 中较慢的 `to_string`。

```cpp
#include <iostream>
#include <cstdio>
using namespace std;

int main(){
    char buf[100];           //必须保证空间足够大！

    int a=123;
    double b=3.14;

    //把a和b格式化写入buf
    sprintf(buf,"a的值是%d，b的值是%.2f",a,b);

    cout<<buf<<endl;
    return 0;
}
```

**2. 进阶用法：字符串拼接**

```cpp
char dir[100];
int year=2023;
int month=10;
int day=27;

//快速拼接文件路径
sprintf(dir,"C:\\logs\\%d-%02d-%02d\\log.txt",year,month,day);
// 注意 %02d 表示不足2位前面补0
// dir 的内容变成了: "C:\logs\2023-10-27\log.txt"
```

> ⚠️ **极度危险警告（缓冲区溢出）：**<br /> >`sprintf` 不会检查目标数组 `str` 的大小。如果生成的字符串超过了数组大小，会导致内存越界，程序崩溃。<br /> >**安全替代方案**：在 C++11 及以后，强烈建议使用 `snprintf`：

```cpp
snprintf(buf, sizeof(buf), "%d", a); // 最多写入 sizeof(buf)-1 个字符，安全！
```

<br />

**二、 sscanf：从字符串中“读”取数据（格式化输入）**<br />
**作用**：按照指定的格式，从一个字符串中提取出各种类型的数据存入变量。<br />

**函数签名：**

```cpp
int sscanf(const char *str, const char *format, ...);
```

- **str**：源字符串（被解析的数据源）。
- **format**：格式控制符。
- **...**：接收数据的变量地址（要加 &）。
- **返回值**：成功匹配和赋值的变量个数。

**1. 基础用法：字符串转数字**<br />
可以从一段杂乱的文本中精准提取数字。

```cpp
#include <cstdio>
using namespace std;

int main() {
    char str[] = "我今年18岁，身高175.5厘米";

    int age;
    double height;

    // 从 str 中按顺序寻找整数和浮点数
    int count = sscanf(str, "我今年%d岁，身高%lf厘米", &age, &height);

    printf("提取了%d个数据\n", count); // 输出: 提取了2个数据
    printf("年龄:%d, 身高:%.1f\n", age, height); // 输出: 年龄:18, 身高:175.5
    return 0;
}
```

<br />

**2. 算法竞赛/开发中的神器：忽略不需要的内容 (%\*)**<br />
有时候你只想提取字符串中间的某一部分，不想管前面的文字，可以用 `%*s` 或 `%*d` 表示“读取但丢弃”。

<br />

**3. 高级用法：按特定字符分割提取 (%[])**<br />
这是 `sscanf` 最强大、也最容易被忽略的功能。

- `%[a-z]`：只读取小写字母。
- `%[0-9]`：只读取数字（相当于提取出一串连续的数字）。
- `%[^a-z]`：读取非小写字母（遇到小写字母就停止）。

**经典场景：提取字符串中的纯数字**

```cpp
char text[] = "abc123def456gh";
int num1, num2;

// "abc" 被跳过，%d 读取 123 赋给 num1，"def" 被跳过，%d 读取 456 赋给 num2
sscanf(text, "%*[^0-9]%d%*[^0-9]%d", &num1, &num2);

printf("%d %d\n", num1, num2); // 输出: 123 456
```

**解释**：`%*[^0-9]` 的意思是：读取一堆不是数字的字符，然后用 `*` 丢弃它们。接着 `%d` 遇到数字就开始提取。

**总结对比：**<br />
| 函数 | 数据流向 | 核心作用 | 典型场景 |
| :--- | :--- | :--- | :--- |
| `scanf` | 键盘 → 变量 | 从控制台读取输入 | 做题时读入数据 |
| `printf` | 变量 → 屏幕 | 格式化输出到控制台 | 做题时打印结果 |
| `sscanf` | 字符串 → 变量 | 解析字符串，提取特定数据 | 从 “2023-10-27” 提取年月日；从文本提取数字 |
| `sprintf` | 变量 → 字符串 | 把数据拼接成特定格式的字符串 | 数字转字符串；按格式生成文件路径 |
