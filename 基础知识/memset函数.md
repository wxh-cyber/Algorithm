# ``memset``函数

<hr />

``memset`` 是 C/C++ 中一个非常常用的库函数，定义在 ``<cstring>`` (C++) 或 ``<string.h>`` C 头文件中。

它的主要用途是**将一块内存区域按照字节设置为指定的值**。

<br />

## 1.函数原型

```cpp
void* memset(void* ptr, int value, size_t num);
```
- **ptr**: 指向要填充的内存块的起始地址。
- **value**: 要设置的值。注意：该值是以 ``int`` 形式传入，但在填充时会转换为 ``unsigned char``（即只取最低的 8 位）。
- **num**: 要设置的字节数。

<br />

## 2.主要用途

<br />

### A.初始化/清空数组（最常用）

这是 ``memset`` 最常见的用法，特别是在数组或结构体初始化时。

**清空数组（置0）**
```cpp
    int arr[10];
    // 将 arr 数组的所有字节设置为 0
    // 此时 arr 中的每个元素都变为 0
    memset(arr, 0, sizeof(arr));
```

<br />

**清空结构体**
```cpp
    struct Student {
        char name[20];
        int age;
    };

    Student s;
    // 将结构体 s 的所有字节清零
    // name 数组变为空，age 变为 0
    memset(&s, 0, sizeof(s));
```

<br />

### B.字符数组填充

由于 ``memset`` 是按字节操作，它非常适合用于操作字符数组。
```cpp
char str[50];
// 将数组前 10 个字节设置为 'a'
memset(str, 'a', 10);
str[10] = '\0'; // 手动添加结束符
// 结果: str = "aaaaaaaaaa"
```

<br />

## 3.陷阱与注意事项

<br />

### 陷阱一：不要用它给整型数组赋非零值

``memset`` 是**按字节**填充的。对于一个 ``int`` 类型（通常占 4 个字节），如果你想把它赋值为 1，结果会出错。

**错误示例**
```cpp
    int arr[5];
    memset(arr, 1, sizeof(arr)); 
    // 期望：arr 中每个元素变为 1
    // 实际：每个字节变成了 0x01
    // int 是 4 字节，内存布局变成了 0x01010101
    // 十进制结果是 16843009，而不是 1！
```
**特例**：只有赋值为 ``0`` 是安全的，因为 ``0x00000000`` 对于整型依然是 ``0``。赋值为 ``-1`` 通常也是安全的（取决于补码表示，通常是 ``0xFFFFFFFF``）。

<br />

### 陷阱二：不要用于含有非 POD 类型的 C++ 类/结构体

在 C++ 中，如果结构体或类里包含指针、``std::string``、``std::vector`` 等对象，**绝对不要**使用 ``memset``清零。

**危险示例**
```cpp
    struct MyClass {
        std::string name;
    };

    MyClass obj;
    memset(&obj, 0, sizeof(obj)); // 错误！
    // 这会粗暴地把 name 对象内部的指针、容量等信息覆盖为 0，
    // 导致析构时发生内存崩溃或未定义行为。
```
对于包含复杂对象的类，应该使用**构造函数**来初始化。

<br />

### 陷阱三：容易算错大小

一定要确保第三个参数（字节数）没有超过分配的内存大小，否则会产生缓冲区溢出，导致程序崩溃或安全漏洞。使用 ``sizeof(变量名)`` 是最安全的写法。

<br />

## 4.替代方案（现代C++推荐）

虽然 ``memset`` 很高效，但在 C++ 中有更安全、更现代的替代方案：

**1.初始化数组为0：**
```cpp
    int arr[10] = {0}; // 更安全，写法更简洁
```

<br />

**2.填充数值（非字节级）：**

使用 ``std::fill`` 或 ``std::fill_n``（定义在 ``<algorithm>`` 中）：
```cpp
    #include <algorithm>
    int arr[10];
    std::fill(arr, arr + 10, 5); // 将每个元素赋值为 5，正确！ 
```

<br />

## 总结

- **适合场景**：高性能清零（数组、POD 结构体）、字符数组填充。
- **禁止场景**：给整型数组赋非 0 值、操作含有复杂成员（如 string、vector）的类对象。
