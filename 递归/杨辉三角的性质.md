# 杨辉三角的性质

<hr />

## 一、基本性质

杨辉三角（又称帕斯卡三角形）是一个将二项式系数按三角形排列的数表，它在数学和计算机科学中具有广泛的应用。它由数字构成，呈现出**每个数字等于其上方两数之和**的规律。杨辉三角不仅体现了二项式展开系数的分布，还蕴含了丰富的组合数学性质。下面我们总结杨辉三角的几个核心性质，并给出数学公式和代码示例加以说明。

<hr />

### 1.构成规律与组合数表示

**性质描述**： 杨辉三角的第 $n$ 行（从0开始计数）有 $n+1$ 个数字，其中每个数字都可以用组合数表示。具体而言，第 $n$ 行的第 $k$ 个数字等于从 $n$ 个不同元素中选取 $k$ 个元素的组合数，记作 $C_n^k$ 。这一性质是杨辉三角与组合数学的桥梁，揭示了其本质。

$C_n^k = \frac{n!}{k!(n-k)!}$

**公式说明**： 组合数 $C_n^k$ 的定义公式如上，其中 $n!$ 表示 $n$ 的阶乘。杨辉三角的第 $n$ 行恰好对应于 $(1+x)^n$ 展开式中各项的系数。例如，第3行的数字为1, 3, 3, 1，正好是 $(1+x)^3 = 1 + 3x + 3x^2 + x^3$ 的系数。

**代码示例**： 下面是一段C++代码，利用组合数公式计算并输出杨辉三角第 $n$ 行的所有数字：
```c++
#include <iostream>
#include <vector>
using namespace std;

/**
 *  @brief 计算组合数C(n,k)
 *  @param n 总个数
 *  @param k 选择个数
 *  @return 组合数C(n,k)
 */
int combination(int n, int k){
    //利用对称性减少计算量
    if(k>n-k) {
        k=n-k;
    }

    int res=1;
    for(int i=1;i<=k;i++){
        res=res*(n-k+i)/i;
    }

    return res;
}

/**
 *  @brief 获取杨辉三角第n行的所有数字
 *  @param rowIndex 行号（从0开始计数）
 *  @return 杨辉三角第n行的所有数字
 */
vector<int> getRow(int rowIndex){
    vector<int> row;

    for(int j=0;j<=rowIndex;j++){
        row.push_back(combination(rowIndex,j));
    }

    return row;
}

int main(){
    int n=5;

    vector<int> row=getRow(n);
    for(int num:row){
        //输出： 1 5 10 10 5 1（对应杨辉三角第5行的数字）
        cout<<num<<" ";
    }
    cout<<endl;

    return 0;
}
```

### 2.对称性

**性质描述**： 杨辉三角具有**左右对称性**，即第 $n$ 行的第 $k$ 个数字与第 $n-k$ 个数字相等。这一性质源于组合数的对称性：$C_n^k = C_n^{n-k}$。直观上，从 $n$ 个元素中选 $k$ 个元素的组合数，等于选 $n-k$ 个元素的组合数，因为选出的元素和未选出的元素是互补的。

$C_n^k = C_n^{n-k}$

**公式说明**： 上述组合恒等式表达了杨辉三角的对称性。例如，在第4行（数字为1, 4, 6, 4, 1）中，第1个数字1与倒数第1个数字1对称，第2个数字4与倒数第2个数字4对称，第3个数字6居中。

**代码示例**： 我们可以利用对称性来优化杨辉三角的生成。下面是一段C++代码，仅计算第 $n$ 行的前半部分数字，然后利用对称性填充后半部分：
```c++
#include <iostream>
#include <vector>
using namespace std;

/**
 *  @brief 计算组合数C(n,k)
 *  @param n 总个数
 *  @param k 选择个数
 *  @return 组合数C(n,k)
 */
int combination(int n, int k){
    //利用对称性减少计算量
    if(k>n-k) {
        k=n-k;
    }

    int res=1;
    for(int i=1;i<=k;i++){
        res=res*(n-k+i)/i;
    }

    return res;
}

/**
 *  @brief 获取杨辉三角第n行的所有数字
 *  @param rowIndex 行号（从0开始计数）
 *  @return 杨辉三角第n行的所有数字
 */
vector<int> getRowOptimized(int rowIndex){
    vector<int> row(rowIndex+1,1);

    for(int j=0;j<=rowIndex/2;j++){
        //计算前半部分的组合数，并利用对称性填充后半部分
        row[j]=row[rowIndex-j]=combination(rowIndex,j);
    }

    return row;
}

int main(){
    int n=6;

    vector<int> row=getRowOptimized(n);
    for(int num:row){
        //输出： 1 6 15 20 15 6 1（对应杨辉三角第6行的数字）
        cout<<num<<" ";
    }
    cout<<endl;

    return 0;
}
```

### 3.行和性质

**性质描述**： 杨辉三角每一行的数字之和等于 $2$ 的该行行数次幂。也就是说，第 $n$ 行（从0开始计数）的所有数字相加等于 $2^n$。这一性质可以通过二项式定理得到解释：令 $(1+1)^n = 2^n$，展开后正好是第 $n$行各数字之和。

$\sum_{k=0}^{n} C_n^k = 2^n$

**公式说明**： 上述公式表示第 $n$ 行所有组合数之和等于 $2^n$。例如，第3行的数字为1, 3, 3, 1，其和为 $1+3+3+1=8=2^3$，验证了该性质。

**代码示例**： 下面是一段C++代码，计算并输出杨辉三角第 $n$ 行所有数字之和，验证其等于 $2^n$：
```c++
#include <iostream>
#include <vector>
#include <cmath>
using namespace std;

int rowSum(int n){
    return pow(2,n);
}

int main(){
    int n=5;
    //输出： 第5行数字之和: 32（因为 2^5 = 32）
    cout << "第" << n << "行数字之和: " << rowSum(n) << endl;

    return 0;
}
```

### 4.斜线性质与斐波那契数列

**性质描述**： 杨辉三角中隐藏着**斐波那契数列**的踪迹。如果将杨辉三角左对齐，并沿特定斜线方向求和，会发现这些和构成斐波那契数列。具体而言，从杨辉三角的第2行第2个数字开始，沿向右下方的斜线相加，得到的和恰好是斐波那契数列的项。斐波那契数列定义为：$F_1=1, F_2=1$，且 $F_n = F_{n-1} + F_{n-2}$（$n \geq 3$）。

$F_n = \sum_{k=0}^{\lfloor\frac{n-1}{2}\rfloor} C_{n-1-k}^k$

**公式说明**： 上述公式表示第 $n$ 个斐波那契数等于杨辉三角中某条斜线上的数字之和。例如，$F_6 = 8$，可以验证杨辉三角中相应的斜线和确实等于8。这一性质揭示了杨辉三角与斐波那契数列之间的深刻联系。

**代码示例**： 下面是一段C++代码，通过遍历杨辉三角的斜线来计算前若干个斐波那契数：
```c++
#include <iostream>
#include <vector>
using namespace std;

vector<vector<int>> generatePascal(int n) {
    vector<vector<int>> triangle(n);

    for (int i = 0; i < n; ++i) {
        triangle[i].resize(i + 1, 1);

        for (int j = 1; j < i; ++j) {
            triangle[i][j] = triangle[i-1][j-1] + triangle[i-1][j];
        }
    }

    return triangle;
}
```
