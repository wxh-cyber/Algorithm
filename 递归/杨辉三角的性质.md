# 杨辉三角的性质

<hr />

## 一、基本性质

杨辉三角（又称帕斯卡三角形）是一个将二项式系数按三角形排列的数表，它在数学和计算机科学中具有广泛的应用。它由数字构成，呈现出**每个数字等于其上方两数之和**的规律。杨辉三角不仅体现了二项式展开系数的分布，还蕴含了丰富的组合数学性质。下面我们总结杨辉三角的几个核心性质，并给出数学公式和代码示例加以说明。

<hr />

### 1.构成规律与组合数表示

**性质描述**： 杨辉三角的第 $n$ 行（从0开始计数）有 $n+1$ 个数字，其中每个数字都可以用组合数表示。具体而言，第 $n$ 行的第 $k$ 个数字等于从 $n$ 个不同元素中选取 $k$ 个元素的组合数，记作 $C_n^k$ 或 $\binom{n}{k}$1。这一性质是杨辉三角与组合数学的桥梁，揭示了其本质。

```latex
$C_n^k = \frac{n!}{k!(n-k)!}$
```
